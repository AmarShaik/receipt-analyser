JUDGE FEEDBACK LIKELY QUESTIONS:

Q: "How does the AI extract data from receipts?"
A: Sends receipt image to Google Gemini API, which uses vision AI to:
   - Read merchant name and location
   - Extract transaction date and time
   - Identify individual items and prices
   - Calculate totals and tax
   - Return as structured JSON

Q: "Can the app work without an API key?"
A: Yes! Demo shows all features without API key.
   To upload REAL receipts, need API key (free from Google).

Q: "Is the data truly private?"
A: Completely private! LocalStorage keeps data in browser.
   Only image sent to Gemini is the receipt being analyzed.

Q: "Can this handle different receipt formats?"
A: Yes, Gemini API handles various formats (grocery, restaurants, gas, etc.)

Q: "What about mobile?"
A: Fully responsive. Works on phones, tablets, and desktop.
